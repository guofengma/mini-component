# mini-component
微信小程序组件集合

#### 目录结构

* component-----------------------------------组件总目录
   * 目录一------------------------------------组件一
   * 目录二------------------------------------组件二
* component_index-----------------------------使用组件demo主页面_使用说明!!!
   * 目录一------------------------------------组件一demo
   * 目录二------------------------------------组件二demo
   * component_index.wxml---------------------小程序页面 通过这个页面汇集组件demo的调用 
* utils---------------------------------------全局工具方法
   * web.js-----------------------------------js常用方法
   
>      组件使用说明在对应组件demo的wxml页面

#### 如何使用 将demo弄进微信小程序
 
  1.直接创建clone此项目
  2.创建小程序
  3.直接导入即可
  ###### 注意 

       图片上传 需要用到  app.js 中的 imgageUploadUrl属性  所以需要在app.js中添加此属性

       app.js 需有userInfo属性
